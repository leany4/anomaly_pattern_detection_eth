# Курсовая: Обнаружение паттернов в транзакциях Ethereum

## Что сделано
1. Анализ двух датасетов (транзакции + адреса)
2. Обучение CatBoost как baseline (F1 = 0.91)
3. Построение GNN для графового анализа
4. Сравнение подходов

## Структура
- `/notebooks` – весь код по этапам 
- `/notebooks/datasets` – исходные датасеты

## Ключевые выводы
- CatBoost лучше на отдельных транзакциях
- GNN выявляет сетевые паттерны
